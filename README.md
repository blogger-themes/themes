## blogger-themes

A monorepo containing modern and experimental open-source Blogger themes built with modern frontend technologies.

The goal of blogger-themes is to explore better developer experience, improved architecture, reusable tooling, and modern rendering techniques for Blogger theme development.

> [!WARNING]
> Most projects inside this monorepo are experimental and may not yet be production-ready.

## Goals

- Modernize Blogger theme development
- Create reusable tooling and shared packages
- Experiment with rendering and hydration strategies
- Improve developer experience for Blogger themes
- Explore PWA support and modern frontend capabilities
- Build open-source Blogger themes with modern architecture

## Features

- ⚛️ Modern frontend stack
- 📦 Monorepo-powered architecture
- 🧩 Shared packages and utilities
- 🔥 TypeScript support
- ⚡ Fast development workflow with Vite
- 🛠️ Reusable UI and runtime packages

## Philosophy

`blogger-themes` focuses on experimenting with modern development workflows while maintaining compatibility with the Blogger platform.

The project prioritizes:

- Maintainability
- Reusability
- Performance
- Developer experience
- Open-source experimentation
